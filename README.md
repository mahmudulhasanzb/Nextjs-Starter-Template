## ✨ Features

- ⚡ **Next.js 15** with App Router
- 🎨 **Tailwind CSS v4** for fast styling
- 🧩 **HeroUI v3** beautiful components
- 📁 Clean and scalable folder structure
- 🔧 **ESLint + Prettier** pre-configured
- 🧩 **Not-Found.jsx, Error.jsx, Loading.jsx** pre-configured
- 🚀 One-command project setup

## 📦 Getting Started

### Option 1 — One Command Setup (Recommended)

```bash
npx create-next-app@latest my-app -e https://github.com/mahmudulhasanzb/Nextjs-Starter-Template
```

```bash
cd my-app
npm run dev
```

### Option 2 — Clone Manually

```bash
git clone https://github.com/mahmudulhasanzb/Nextjs-Starter-Template.git my-app
cd my-app
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result. 🎉

## 📁 Folder Structure

```bash
src/
├── app/
│   ├── layout.js         # Root layout (Navbar + Footer)
│   ├── page.js           # Home page
│   └── globals.css       # Global styles
├── components/
│   ├── layout/
│   │   ├── Navbar.jsx    # Navigation bar
│   │   └── Footer.jsx    # Footer
│   └── ui/               # Reusable UI components
├── hooks/                # Custom hooks (coming in v2)
└── lib/                  # Utility functions
```
