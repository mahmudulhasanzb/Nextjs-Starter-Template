# Next.js Starter Template

A clean and minimal Next.js starter template for fast project setup.

## Tech Stack

- Next.js (App Router)
- Tailwind CSS v4
- HeroUI
- ESLint + Prettier

## Getting Started

1. Clone the repo
2. Install dependencies — npm install
3. Copy env file — cp .env.example .env.local
4. Run dev server — npm run dev
